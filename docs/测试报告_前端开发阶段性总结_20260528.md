# 前端开发阶段性总结

## 一、项目概述

本项目是一个基于 Vue 3 + TypeScript 的文言文学习平台，旨在为学生提供文言文学习的互动体验。项目采用现代化的前端技术栈，包含完整的用户认证系统、答题进度管理、多角色朗读等核心功能。

## 三、数据流向与依赖关系

### 3.1 架构层次

```
┌─────────────────────────────────────┐
│          视图层 (Views)             │
│   HomeView / StepOneView / ...     │
│   使用: useNavigation, authStore   │
├─────────────────────────────────────┤
│          组件层 (Components)        │
│   WordList / MultiRoleReading      │
│   使用: useDataLoader, adapters    │
├─────────────────────────────────────┤
│        组合式层 (Composables)       │
│   useNavigation, useQuizProgress   │
│   使用: vue-router, Pinia          │
├─────────────────────────────────────┤
│         状态层 (Stores)             │
│   authStore, studentStore          │
│   使用: Pinia, localStorage        │
├─────────────────────────────────────┤
│          工具层 (Utils)             │
│   api.ts, wenUtils.ts              │
├─────────────────────────────────────┤
│          数据层 (public/data)       │
│   JSON文件 / 静态资源              │
└─────────────────────────────────────┘
```

### 3.2 核心数据流

1. **页面导航**：`views` → `useNavigation` → `vue-router` → 更新URL
2. **数据加载**：`components` → `useDataLoader` → `fetch` → `adapters` → 组件渲染
3. **状态管理**：`views/components` → `Pinia stores` → `localStorage`
4. **API请求**：`utils/api.ts` → 自动携带JWT → 后端接口
