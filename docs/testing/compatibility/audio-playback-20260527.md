## 六、播放状态管理架构

### 6.1 单一真相来源设计

```typescript
// usePlaybackControl.ts - 唯一的播放状态
const isPlaying = ref(false)

// 所有按钮共享同一状态
export function usePlaybackControl(audioRef, options) {
  return {
    isPlaying,      // 唯一的播放状态
    togglePlay,     // 切换播放/暂停
    play,           // 开始播放
    pause,          // 暂停播放
    seekTo,         // 跳转到指定时间
  }
}
```

### 6.2 状态同步规则

| 操作 | 状态变化 | 同步行为 |
|-----|---------|---------|
| 主按钮点击 | isPlaying = !isPlaying | 所有子按钮同步更新 |
| 子按钮播放 | isPlaying = true | 主按钮显示播放状态 |
| 子按钮暂停(当前段落) | isPlaying = false | 主按钮显示暂停状态 |
| 音频播放结束 | isPlaying = false | 所有按钮同步更新 |
