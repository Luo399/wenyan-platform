## 五、兼容性限制说明

### 5.1 IE浏览器支持

**当前状态**: 不支持

**原因分析**:

- Vite配置未包含 `@vitejs/plugin-legacy` 插件
- Vue 3原生不支持IE 11
- ES6+语法无法在IE中运行

**解决方案**:

```bash
# 如需支持IE，安装legacy插件
npm install @vitejs/plugin-legacy
```

### 5.2 360浏览器兼容模式

**当前状态**: 部分支持

**原因分析**:

- 兼容模式使用IE内核
- 图标CDN在IE内核下显示异常

**解决方案**:

- 添加meta标签强制极速模式
- 考虑将图标资源本地化

### 5.3 媒体自动播放限制

**当前状态**: 所有浏览器均受限

**原因分析**:

- 浏览器安全策略限制自动播放
- 需要用户交互后才能播放媒体

**解决方案**:

- 在页面添加明显的播放按钮
- 添加自动播放失败的降级提示
